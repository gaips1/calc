# Калькулятор
Консольный калькулятор на Python. Поддерживает базовые арифметические операции, работу с памятью и сохранение истории вычислений. Точность расчётов достигается благодаря использованию модуля Decimal. d_sv

---

# Запуск

## Запуск калькулятора
`python calc.py`

## Запуск автотестов
`pip install -r requirements.txt`  
`pytest -v test_calc.py`  

После установки зависимостей возможно придётся перезапустить терминал.
